# Button Challenge

Welcome to my **Button Animation Challenge**! 🚀  

This project is a **series of interactive button micro-interactions** built with **Tailwind CSS**. The goal of this challenge is to explore **creative, modern UI interactions** for buttons, keeping them subtle, fun, and engaging.

---

## 🌟 Features

- **Magnetic Glow Button:** Follows cursor for a premium interactive feel.
- **Gentle Bounce Button:** Smooth hover-triggered bounce animation.
- **Fully Responsive:** Works on mobile, tablet, and desktop.
- **Tailwind CSS:** Lightweight, modern, and easily customizable.
- **Organized by Challenge Day:** Each button micro-interaction is in its own folder for clarity.
